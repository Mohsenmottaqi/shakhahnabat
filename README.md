### Sections

2. **Menu** - Categorized food items (Afghan, Pizza, Fast Food, Healthy)
3. **Specialties** - Featured Afghan dishes
4. **About** - Restaurant story and statistics
5. **Contact** - Contact form and information
6. **Footer** - Links and credits

### Language Toggle
The website automatically switches between Persian (RTL) and English (LTR) layouts:
- **Hero Title**: Restaurant name color changes
- **About Section**: Image placement (FA: left, EN: right)
- **Contact Section**: Form placement (FA: right, EN: left)
- **Form Fields**: Text alignment matches language direction
